Pour lancer ce programme il faut avoir la bibliothèque SFML installé sur son système linux

Pour installer SFML, lancer la commande suivante: 

	-> sudo apt-get install libsfml-dev



Pour compiler le programme, allez dans le dossier Debug et lancer la commande: 
	
	-> make all


Pour lancer le programme compilé, rester dans le dossier Debug et lancer la commande:

	-> ./dominos_cpp
